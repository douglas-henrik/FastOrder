# FastOrder

Sistema de gerenciamento de pedidos para lanchonetes desenvolvido em Java.

## Funcionalidades

- Cadastro de mesas
- Cadastro de produtos
- Exibição do cardápio
- Abertura de pedidos
- Adição de itens ao pedido
- Visualização dos pedidos
- Fechamento de pedidos

## Tecnologias

- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- Enum
- Scanner

## Estrutura do projeto

```
src
├── enums
├── main
├── model
└── service
```

## Conceitos utilizados

- Encapsulamento
- Composição
- Enum
- Coleções (ArrayList)
- Organização em pacotes
- Separação de responsabilidades
